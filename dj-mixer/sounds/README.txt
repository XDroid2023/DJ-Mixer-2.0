Sound effects should be placed in this directory with the following names:
- scratch.mp3: DJ scratch sound effect
- airhorn.mp3: Classic airhorn sound
- drumroll.mp3: Drum roll effect
- transition.mp3: DJ transition effect
- bassdrop.mp3: Bass drop effect
- cymbal.mp3: Cymbal crash effect

You can download free sound effects from websites like:
- https://freesound.org
- https://soundbible.com
- https://pixabay.com/sound-effects/

Make sure to rename the downloaded files to match the expected filenames above.
